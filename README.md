# Chinook Business Analysis

## 🇬🇧 English

### Project Overview

This project analyzes the Chinook sample database using SQL and Power BI.

The objective is to transform transactional data into meaningful business insights through SQL querying, data aggregation, and interactive visualizations. The project replicates a real-world business analysis scenario by exploring sales performance, customer behavior, and revenue drivers.

### Business Questions

The analysis focuses on answering the following questions:

- Which countries generate the highest revenue?
- Who are the top customers by spending?
- Which artists generate the highest revenue?
- What are the best-selling albums?
- Which music genres contribute the most to revenue?
- What are the main sales KPIs across different markets?

### Descrizione del Progetto

Questo progetto analizza il database Chinook utilizzando SQL e Power BI.

L'obiettivo è trasformare dati transazionali in informazioni utili attraverso interrogazioni SQL, aggregazioni e visualizzazioni interattive. Il progetto simula un caso reale di business analysis, analizzando performance di vendita, comportamento dei clienti e principali driver di fatturato.
